Setup Guide

Follow these steps to go fully live. Everything is free and uses Google's tools. Submissions auto-save to a Google Sheet only your executives can access, and you get birthday email alerts every morning.

Step 01 — Create Your Google Form

Build the member registration form

Go to forms.google.com → New Form → Title it "CMDA 2k29 UI — Class Directory". Add a description: "Caring for the whole man: spirit, soul and body — 1 Thess. 5:23". Add these fields in this exact order:

  • Full Name — Short answer (Required)
  • Nickname / Alias — Short answer
  • Date of Birth — Date (Required)
  • Gender — Multiple choice (Male / Female / Prefer not to say)
  • Marital Status — Dropdown
  • Phone Number — Short answer (Required)
  • Email Address — Short answer (Required)
  • WhatsApp Number — Short answer
  • Emergency Contact (Name & Phone) — Short answer
  • State of Origin — Dropdown (all 36 states + FCT)
  • LGA of Origin — Short answer
  • Current City / Location — Short answer
  • Matric Number — Short answer
  • Hall of Residence — Short answer
  • Specialty / Career Aspiration — Paragraph
  • Fun Fact / Hobbies — Paragraph
  • Profile Photo — File upload (set max file size to 5MB, image files only)
💡 Tip: After creating the form, click the Settings tab → uncheck "Collect email addresses" if you don't want Google accounts required, so any member with a link can fill it.
Step 02 — Link to Google Sheets (Auto-save)

Every submission saves automatically

In your form, click the Responses tab → click the green Sheets icon → Create a new spreadsheet. Name it "CMDA 2k29 UI — Directory (PRIVATE)".

Now click Share on the spreadsheet → add your executives' Gmail addresses as Editors. They can view/edit anytime from any device. Never share the sheet with non-executives.

✅ From this moment, every form submission appears as a new row in the sheet — automatically, in real time, with no extra steps.
Step 03 — Birthday Alert Script (Runs Every Morning)

Get emailed 1 day before every birthday

In your Google Sheet: click Extensions → Apps Script. Delete all existing code and paste the script below exactly. Then click Run → setup() once to activate the daily trigger.

// ============================================================ // CMDA 2k29 UI — Birthday Alert Script // Sends an email 1 day before a member's birthday, every morning // ============================================================ function checkBirthdays() { var sheet = SpreadsheetApp.getActiveSpreadsheet() .getSheets()[0]; var data = sheet.getDataRange().getValues(); var today = new Date(); var tomorrow = new Date(today); tomorrow.setDate(today.getDate() + 1); var tmMonth = tomorrow.getMonth() + 1; // 1-12 var tmDay = tomorrow.getDate(); // ⚠️ ADJUST THIS INDEX if Date of Birth is not in column C (index 2) // Column A = index 0, B = 1, C = 2, D = 3 ... var DOB_COL = 2; // Date of Birth column var NAME_COL = 0; // Full Name column var PHONE_COL= 5; // Phone Number column var EMAIL_COL= 6; // Email Address column var celebrants = []; for (var i = 1; i < data.length; i++) { var row = data[i]; var raw = row[DOB_COL]; if (!raw) continue; var dob = new Date(raw); if (isNaN(dob.getTime())) continue; if ((dob.getMonth() + 1) === tmMonth && dob.getDate() === tmDay) { celebrants.push({ name: row[NAME_COL] || "(No name)", phone: row[PHONE_COL] || "—", email: row[EMAIL_COL] || "—" }); } } if (celebrants.length === 0) return; // No birthday tomorrow var dateStr = tomorrow.toDateString(); var subject = "🎂 CMDA 2k29 UI — Birthday Tomorrow! (" + dateStr + ")"; var body = "Greetings General Secretary,\n\n" + "The following classmate(s) celebrate their birthday TOMORROW, " + dateStr + ":\n\n"; celebrants.forEach(function(m) { body += "━━━━━━━━━━━━━━━━━━━━\n"; body += "👤 " + m.name + "\n"; body += "📞 " + m.phone + "\n"; body += "✉️ " + m.email + "\n\n"; }); body += "━━━━━━━━━━━━━━━━━━━━\n"; body += "Remember to celebrate them on behalf of the class!\n\n"; body += "— CMDA 2k29 UI Automated System\n"; body += "\"Caring for the whole man: spirit, soul and body\" — 1 Thess. 5:23"; MailApp.sendEmail({ to: Session.getActiveUser().getEmail(), subject: subject, body: body }); } // ─── Run setup() ONCE to activate the daily trigger ──────── function setup() { // Remove existing triggers first (avoid duplicates) var triggers = ScriptApp.getProjectTriggers(); triggers.forEach(function(t) { ScriptApp.deleteTrigger(t); }); // Create new daily trigger at 7:00 AM ScriptApp.newTrigger('checkBirthdays') .timeBased() .everyDays(1) .atHour(7) .create(); Logger.log("✅ Birthday alert trigger set! Runs daily at 7am."); }
⚠️ Important: The column index numbers (DOB_COL, NAME_COL, etc.) match the order of fields in your Google Form. If you add or remove fields, update these numbers. Column A = 0, B = 1, C = 2, D = 3, and so on.
Step 04 — Grant Permissions & Activate

One-time activation

  • After pasting the script, click Save (💾)
  • Select setup from the function dropdown → click ▶ Run
  • Google will ask for permission — click Review permissions → Allow
  • You'll see "Birthday alert trigger set!" in the Logs panel
  • That's it — you'll receive an email every morning a birthday is due the next day
Step 05 — Download the Directory Anytime

Export member data as CSV or PDF

From your Google Sheet anytime: File → Download → CSV for a spreadsheet, or File → Download → PDF for a printable directory. You can also use the Download CSV and Download PDF buttons in the Members tab of this page for data entered through this prototype.

Step 06 — Share Your Form Link

Get members to fill in their details

In Google Forms, click Send → 🔗 Link icon → Copy. Share it in your CMDA WhatsApp group. For a cleaner link, use bit.ly to shorten it — e.g. bit.ly/cmda2k29ui.

You can also paste the link in the Join Directory tab of this page — it will show members a direct button to your form.
